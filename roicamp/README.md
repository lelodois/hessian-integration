# Roicamp 

É uma api REST que disponibiliza serviços de campanha, usuario e associação dos mesmos.

## Arquitetura

* RestAPI
* Java 8
* Maven
* Spring boot / Mvc / Event
* Hibernate / Listener
* Swagger
* H2 / H2 console
* Junit
* Mockito
